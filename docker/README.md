# docker/

サンドボックス環境の Docker 構成ファイルを格納しています。

## ファイル構成

| ファイル | 説明 |
|---------|------|
| `Dockerfile.sandbox` | サンドボックス実行環境の Docker イメージ定義 |
| `whitelist.yml` | サンドボックス内で許可するコマンド・操作のホワイトリスト |
