**参照先**：[`docs/手順書/運用保守手順書.md`](../手順書/運用保守手順書.md)

### 統合の理由

- 本ランブックと運用保守手順書は、対象読者（Owner / Maintainer）と内容（日常運用操作）が重複していた
- 運用保守手順書がランブックの上位互換として、より包括的な手順を提供していた
- 2つの文書の並存により更新漏れ・矛盾が発生するリスクがあった
