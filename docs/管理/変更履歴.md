# 変更履歴 (CHANGELOG)

> 📖 **この文書について**  
> この文書は、本システムの**変更内容を時系列で記録した履歴**です。  
> たとえるなら、「アプリのアップデート履歴」のようなものです。どのバージョンで何が追加・修正されたかを確認できます。

### 🔰 この文書で使われる主要な用語

| 用語 | かんたんな説明 |
|------|-------------|
| **Unreleased** | まだリリースされていない開発中の変更 |
| **Added** | 新しく追加された機能 |
| **Fixed** | 修正されたバグや不具合 |
| **セマンティックバージョニング** | バージョン番号の付け方のルール（メジャー.マイナー.パッチ） |
| **ruff** | Pythonコードの書き方をチェックするツール |
| **lint** | コードの品質チェックのこと |
