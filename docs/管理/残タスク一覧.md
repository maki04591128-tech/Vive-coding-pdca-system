## 🔴 高優先度（機能欠落）

### 1. CI アダプターの `get_status()` API 呼び出し未実装

| 項目 | 詳細 |
|------|------|
| **ファイル** | `src/vibe_pdca/engine/ci_adapter.py` |
| **該当箇所** | `GitHubActionsAdapter.get_status()` (L181)、`GitLabCIAdapter.get_status()` (L253) |
| **現状** | `raise NotImplementedError("... API 呼び出しは未実装です。")` |
| **影響** | 外部CIサービスからのビルド結果自動取得が不可能。`normalize_result()` による手動データ投入は可能 |
| **対応案** | `httpx` による GitHub Actions API / GitLab CI API の実装。認証トークンは設定ファイル or 環境変数から取得 |
| **要件定義参照** | §9.3（CI統合） |

### 2. GUIテストの CI 除外問題

| 項目 | 詳細 |
|------|------|
| **ファイル** | `tests/test_gui.py`（232行） |
| **現状** | `flet` モジュール未インストール時に `ModuleNotFoundError` でテスト収集失敗 |
| **影響** | GUI関連のリグレッションがCIで検出されない |
| **対応案** | (A) `pytest.importorskip("flet")` による条件スキップ、(B) CI に `pip install -e ".[dev,gui]"` 追加 |
