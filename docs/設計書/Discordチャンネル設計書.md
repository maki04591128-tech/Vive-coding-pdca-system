## 2. インフラ保守カテゴリ（OpenClaw）

### 🔴 hw-critical（重要度：高）

**投稿者**: OpenClaw のみ  
**メンション**: 全投稿で `@Owner` を付ける  
**対応期限**: 通知から **1時間以内** に人間が確認する  
**削除ルール**: 削除しない（障害記録として永続保持）

投稿される内容:

```
発火条件                              重大理由
────────────────────────────────────────────────────────
GPU温度 ≥ 95℃                        熱暴走・強制シャットダウンの危険
GPU ECC Uncorrected Error 検出        GPUの物理的損傷・データ化け
NVMe S.M.A.R.T. 異常（代替処理済み）  ディスク故障の前兆
UPS残量 ≤ 15%（かつ停電中）           数分以内にシャットダウンの必要
vLLMプロセス再起動3回失敗             手動介入が必要な障害
NIC リンクダウン（30秒以上）           サーバーがネットワーク孤立
ディスク使用率 ≥ 95%                  残り数十GBでシステム停止の危険
```

通知フォーマット:

```
🚨 【HW CRITICAL】@Owner
━━━━━━━━━━━━━━━━━━━━━━━
検知時刻 : 2026-03-04 14:32 JST
内容     : GPU0 ECC Uncorrectedエラー検出（3件）
場所     : GPU0 (A100 40GB PCIe スロット1)
自動対処 : なし（物理的対処が必要）
対応期限 : 1時間以内に確認をお願いします
━━━━━━━━━━━━━━━━━━━━━━━
推奨アクション:
  1. nvidia-smi -q -d ECC でエラー詳細を確認
  2. エラーが増加傾向にある場合はGPU交換を検討
  3. PDCAシステムへの影響が懸念される場合は手動停止を検討
```

## 5. メッセージ削除の実装方法

### OpenClaw 側（hw-status の1週間削除）

```python
# openclaw_skills/discord_cleanup.py
# cronで毎日1回実行

import httpx
import time
from datetime import datetime, timedelta

DISCORD_BOT_TOKEN = "..."  # 環境変数から取得
HW_STATUS_CHANNEL_ID = "..."

def cleanup_old_messages(channel_id: str, days: int):
    """指定チャンネルの古いメッセージを削除"""
    threshold = datetime.utcnow() - timedelta(days=days)
    
    # メッセージ一覧を取得
    headers = {"Authorization": f"Bot {DISCORD_BOT_TOKEN}"}
    messages = httpx.get(
        f"https://discord.com/api/v10/channels/{channel_id}/messages?limit=100",
        headers=headers
    ).json()
    
    deleted = 0
    for msg in messages:
        # タイムスタンプを確認
        created_at = datetime.fromisoformat(msg["timestamp"].replace("Z", "+00:00"))
        if created_at.replace(tzinfo=None) < threshold:
            httpx.delete(
                f"https://discord.com/api/v10/channels/{channel_id}/messages/{msg['id']}",
                headers=headers
            )
            deleted += 1
            time.sleep(0.5)  # レート制限対策
    
    return deleted

# 実行（毎日3:00 cronで呼び出す）
cleanup_old_messages(HW_STATUS_CHANNEL_ID, days=7)
```
