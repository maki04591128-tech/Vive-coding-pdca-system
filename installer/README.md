# installer/

配布用インストーラーのビルドモジュールを格納しています。

## ファイル構成

| ファイル | 説明 |
|---------|------|
| `__init__.py` | パッケージ初期化 |
| `build.py` | PyInstaller を使用した EXE ビルドスクリプト |
| `downloader.py` | モデル・依存ファイルのダウンローダー |
| `installer_config.yml` | インストーラーのビルド設定 |
