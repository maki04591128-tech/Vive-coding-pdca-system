## 脆弱性の報告方法

セキュリティ上の問題を発見された場合は、**公開 Issue を作成せず**、
[GitHub Security Advisory](https://github.com/maki04591128-tech/Vive-coding-pdca-system/security/advisories/new) から非公開で報告してください。

48時間以内に初期回答を行います。
