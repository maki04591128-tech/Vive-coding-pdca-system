# config/

システムの設定ファイルを格納しています。

## ディレクトリ構成

| パス | 説明 |
|------|------|
| `default.yml` | グローバルデフォルト設定 |
| `environments/` | 環境別設定（`dev.yml`, `prod.yml`） |
| `i18n/` | 国際化（多言語）リソース |
| `prompts/` | ペルソナ別プロンプトテンプレート |

## 設定の優先順位

```
環境変数 > プロジェクト固有設定(.vibe-pdca/config.yml) > 環境別設定 > グローバルデフォルト
```
